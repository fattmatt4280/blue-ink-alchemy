

## Add TikTok Domain Verification File

Create a single file in the `public` folder so TikTok can verify domain ownership.

### File to create

**`public/tiktok-developers-site-verification.txt`**

Contents:
```
tiktok-developers-site-verification=3OO09t6aDEW0u33G8uEnvKgUOeF5hKJT
```

This will be accessible at `https://bluedreambudder.lovable.app/tiktok-developers-site-verification.txt` after publishing.
